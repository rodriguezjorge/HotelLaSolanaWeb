# HotelLaSolanaWeb

Sitio web estático y responsivo para el Hotel La Solana.

## Estructura del proyecto

- `index.html` – Página principal del hotel (inicio, habitaciones, servicios, galería, ubicación y contacto).
- `styles.css` – Estilos del sitio, con diseño moderno y adaptable a móviles.
- `script.js` – Funcionalidades básicas (menú móvil, scroll suave, etc.).
- `package.json` – Configuración mínima de Node para poder levantar un servidor de desarrollo.

## Requisitos

- Node.js y npm instalados en tu equipo (para usar el servidor de desarrollo).

## Cómo ejecutar el proyecto

1. Instala las dependencias:

   ```bash
   npm install
   ```

2. Inicia el servidor de desarrollo:

   ```bash
   npm run dev
   ```

3. Abre en tu navegador:

   `http://localhost:8080`

Si no quieres usar Node, también puedes abrir directamente el archivo `index.html` con tu navegador (doble clic sobre el archivo).

## Personalización

- Reemplaza textos, fotos y enlaces con la información real del hotel.
- Sustituye las imágenes de ejemplo por fotografías extraídas de tu propio banco de imágenes del hotel.
